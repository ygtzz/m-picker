# Picker

a component to picker data on mobile device

## Installation
```bash
npm i my-picker -S
```

## Usage
```javascript
import {Picker} from 'my-picker';

new Picker({
    con:'.picker'
})
```   

## Params

Parameter | Type |Default| Description
--------- | ---- | ------|-----------
con    | `string` |  | picker's container
